Transportation Optimizer Minimize group transportation time with dynamic conditions

Problem Description

We have n individuals and a single vehicle with capacity k. Each person has a base crossing time, and the environment goes through m stages that multiply these times. The optimizer searches for the sequence of forward and return trips that transports everyone in the minimum possible total time (or returns -1 if it is impossible).
